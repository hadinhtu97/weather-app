# Weather App

## Description
A page that shows weather information of current location

## Use 
```
git clone https://github.com/hadinhtu97/weather-app.git
cd weather-app
npm install
npm run build
npm run start
```
## Api
```
GET request to : []\api\weather?lat={lattitude}&lon={longitude}
```
